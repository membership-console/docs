本プロジェクトでは RBAC（ロールベースアクセス制御）を導入しています。

システムが提供するロールを組み合わせたユーザグループ（役職のようなもの）を作成し、それをユーザごとに割り当てることでアクセス制御を実現します。

## ロールリスト

| ID  | 名前                                    |
| --- | --------------------------------------- |
| 0   | IAM の管理者（IAM_ADMIN）               |
| 1   | 会計システムの管理者（PAYMASTER_ADMIN） |
| 2   | リマインダーの管理者（REMINDER_ADMIN）  |
